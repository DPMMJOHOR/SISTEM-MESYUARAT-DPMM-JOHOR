## How to Create Assets

1. Use the DPMM Johor logo as the base
2. Ensure high resolution (at least 1024x1024 for icons)
3. Use the DPMM brand colors (blue: #2563EB)
4. For adaptive icon, ensure the content is within the safe zone (center 66% of the image)
5. Test the assets on both iOS and Android simulators

## Asset Generation Tools
